## ⚙️ How to Mount an ISO File

Mounting an image tricks the computer into thinking you inserted a real disc into the drive.

1. Open the Magic-ISO-2026 application from your start menu.
2. Click the File menu in the top left corner.
3. Select Open and browse to the location of your ISO file.
4. Click the Mount icon in the toolbar.
5. Open your File Explorer. You will see a new virtual disc drive listed under This PC.
6. Click the drive to view the contents of the image.
